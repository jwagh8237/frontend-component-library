# Frontend Component Library

A collection of reusable and responsive website components built with HTML, CSS, and JavaScript.

## Features

* 50+ reusable components
* 20+ layout variants
* Fully responsive design
* Dark mode support
* Copy code functionality
* Multiple section variants
* Mobile-friendly navigation

## Components

* Navigation
* Hero Sections
* Services
* Features
* Contact
* Blog
* CTA
* Footer
* Other UI Components

## Technologies

* HTML
* CSS
* JavaScript

## Purpose

This project was created to speed up website development by providing reusable frontend components that can be quickly assembled into complete websites.
